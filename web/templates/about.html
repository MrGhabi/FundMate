About FundMate

What is FundMate?

FundMate is a production-ready financial data processor that extracts cash holdings and position data from broker statements using LLM-powered data extraction.

It processes both PDF statements (image-based) and Excel files (structured data) from 12+ different brokers, providing real-time asset valuation with market prices.

Key Features

  • Multi-Broker Support - Processes statements from 12+ major brokers
  • LLM-Powered Extraction - Uses Google Gemini AI for intelligent data extraction
  • Real-Time Pricing - Fetches current market prices via Futu OpenD API
  • Option Contract Handling - Intelligent multiplier logic for US and HK options
  • Multi-Currency Support - Automatic conversion (USD, CNY, HKD)
  • Concurrent Processing - Fast parallel processing of multiple accounts
  • Web Interface - Modern, responsive dashboard for data visualization

Supported Brokers

IB (Interactive Brokers) MOOMOO FUTU MS (Morgan Stanley) GS (Goldman Sachs) LB (Longbridge) TIGER WEBULL SC (Standard Chartered) HSBC USMART FIRST

Technology Stack

Backend: Python 3.10, Flask
AI/ML: Google Gemini 2.5 Pro
Data Processing: Pandas, PyArrow (Parquet)
PDF Processing: pdfplumber, pdf2image
Market Data: Futu OpenD API, akshare
Frontend: Jinja2, Vanilla JavaScript, CSS3

Data Flow

Input PDFs/Excel
    ↓
PDF Processing (Decryption, Page Filtering)
    ↓
LLM Extraction (Structured Data from Images)
    ↓
Price Fetching (Real-time Market Prices)
    ↓
Cross-Broker Optimization (Batch Queries)
    ↓
Output (Parquet/CSV) → Web Interface

Get Started

To start using FundMate:

  1. Ensure Futu OpenD is running on port 11111
  2. Process your broker statements: python src/main.py ./data/statements --date YYYY-MM-DD
  3. View your portfolio data in this web interface
